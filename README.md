# 彼の気持ちタロット 完全版

HTML / CSS / JavaScript だけで動く静的サイトです。

## 追加内容
- 正位置 / 逆位置
- カード枚数9枚
- 悩み選択：復縁 / 片思い / 恋人 / 音信不通
- 占い師ごとにLINE文・サービス名・LINEリンクを差し替え可能
- 過去 / 現在 / 未来 の3枚引き

## 使い方
1. `index.html` を開く
2. 悩みを選ぶ
3. 必要なら「占い師用：LINE導線文を差し替える」を編集
4. 過去 → 現在 → 未来 の順でカードを選ぶ
5. 結果を確認する

Cloudflare Pages / GitHub Pages にそのままアップできます。
